# Usage

<!-- usage -->

```sh-session
$ npm install -g calc-cli
$ calc-cli COMMAND
running command...
$ calc-cli (-v|--version|version)
calc-cli/0.0.1 darwin-x64 node-v10.16.3
$ calc-cli --help [COMMAND]
USAGE
  $ calc-cli COMMAND
...
```
